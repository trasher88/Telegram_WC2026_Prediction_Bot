# Изменения в этой сборке

## Профиль пользователя

- При первом `/start` бот просит ввести имя для таблицы турнира.
- Добавлена команда `/rename` для смены имени.
- В БД добавлены поля `display_name`, `name_set`, `created_at`, `updated_at`.
- Существующая база обновляется автоматически при запуске через `init_db()`.
- Лидерборд, админский список пользователей и рассылки используют `display_name` с fallback на Telegram username / first_name.

## Ежедневные уведомления

- Старые уведомления за 1 час и за 10 минут до каждого матча больше не планируются.
- Добавлены ежедневные cron-уведомления по московскому времени:
  - 13:00 — всем пользователям список матчей игрового дня;
  - 18:00 — только пользователям с недостающими прогнозами;
  - 18:30 — администраторам отчёт по прогнозам.
- Игровой день считается как период с 13:00 МСК текущего дня до 12:59 МСК следующего дня.
- Техническая блокировка прогнозов за 5 минут до матча сохранена.
- Автоматическая lock-рассылка всем пользователям отключена; `/test_lock` в админке по-прежнему может отправить lock-рассылку вручную.
- Добавлена таблица `daily_notifications`, чтобы не дублировать дневные уведомления после рестарта.

## Флаги

- Исправлены флаги Англии и Шотландии на Unicode subdivision flag sequences.
